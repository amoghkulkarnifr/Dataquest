2: Array comparisons

One of the most powerful aspects of NumPy is the ability to make comparisons across an entire array. These comparisons result in Boolean values.

Here's an example with a vector:

xxxxxxxxxx
 
vector = numpy.array([5, 10, 15, 20])
vector == 10

If you'll recall from an earlier mission, the double equals sign (==) compares two values. In the case of NumPy, the second value will be compared to each element in the vector. If the value are equal, True will be returned, and False otherwise.

The above code will result in the vector [False, True, False, False]. This is because only the second element in vector equals 10.

Here's an example with a matrix:

xxxxxxxxxx
 
matrix = numpy.array([
                    [5, 10, 15], 
                    [20, 25, 30],
                    [35, 40, 45]
                 ])
    matrix == 25

This will compare 25 to every element in matrix. The result will be a matrix where elements are True or False:

xxxxxxxxxx
 
[
    [False, False, False], 
    [False, True,  False],
    [False, False, False]
]

The result only contains True in a single position -- where the corresponding element in matrix is 25.

3: Selecting elements

We said earlier that comparisons were very powerful, but it may not have seemed obvious why they were so powerful in the last screen. Their power stems from the fact that you can select elements in arrays using Boolean vectors. This allows us to conditionally select certain elements in vectors, or certain rows in matrices.

Here's an example with a vector:

xxxxxxxxxx
 
vector = numpy.array([5, 10, 15, 20])
equal_to_ten = (vector == 10)
​
print(vector[equal_to_ten])

Step by step, here's what the code above is doing:

  • We create vector.
  • We compare vector to the value 10, which generates a Boolean vector [False, True, False, False]. We assign the result to equal_to_ten.
  • We use equal_to_ten to only select elements in vector where equal_to_ten is True. This results in the vector [10].

We can also select rows in matrices using the same principle:

xxxxxxxxxx
 
matrix = numpy.array([
                [5, 10, 15], 
                [20, 25, 30],
                [35, 40, 45]
             ])
    second_column_25 = (matrix[:,1] == 25)
    print(matrix[second_column_25, :])

Step by step, here's what the code above is doing:

  • We create matrix.
  • We compare the second column of matrix with the value 25. This results in a vector [False, True, False], which we assign to second_column_25.
  • We use second_column_25 to select any rows in matrix where second_column_25 is True.

We end up with the matrix below:

xxxxxxxxxx
 
[
    [20, 25, 30]
]

We selected just a single row from matrix, which has been returned in a new matrix.

4: Comparisons with multiple conditions

In the last screen, we made comparisons with a single condition. We can also perform comparisons with multiple conditions. When we do this, we specify each condition separately, then join them with the & symbol. When constructing a comparison with multiple conditions, it's critical to put each condition in parentheses.

Here's an example with a vector:

xxxxxxxxxx
 
vector = numpy.array([5, 10, 15, 20])
equal_to_ten_and_five = (vector == 10) & (vector == 5)

In the above statement, we have two conditions, (vector == 10) and (vector == 5). By using the & symbol, we indicate that both conditions must be True for the final result to be True. The statement returns [False, False, False, False], because no elements can simultaneously be 10 and 5. Here's a diagram:

We can also use the | symbol to specify that one condition or the other should be True:

xxxxxxxxxx
 
vector = numpy.array([5, 10, 15, 20])
equal_to_ten_or_five = (vector == 10) | (vector == 5)

The code above will result in [True, True, False, False].

5: Replacing values

Another powerful use for comparisons is replacing values in an array. We can conditionally replace certain values with other values.

Here's an example for a vector:

xxxxxxxxxx
 
vector = numpy.array([5, 10, 15, 20])
equal_to_ten_or_five = (vector == 10) | (vector == 5)
vector[equal_to_ten_or_five] = 50
print(vector)

This will result in [50, 50, 15, 20]. Step by step, here's what's happening:

  • We create an array vector.
  • We compare vector to 10 and 5, and generate a vector that is True where vector is equal to either value.
  • We select only the elements in vector where equal_to_ten_or_five is True.
  • We replace the selected values with the value 50.

Here's a diagram:

We can perform the same replacement on a matrix by first using indexing to select a column or row:

            [35, 40, 45]
         ])

    matrix[second_column_25, 1] = 10

The above code will result in:

xxxxxxxxxx
 
[
    [5, 10, 15], 
    [20, 10, 30],
    [35, 40, 45]
 ]

Here's what's happening step by step:

  • We create an array matrix.
  • We compare the second column of matrix to the value 25.
  • We select any values in the second column of matrix where the value is 25.
  • We replace the selected values with 10.

6: Replacing empty strings

We'll want to work with the Display Value column in the future, because it shows how much alcohol people in a country drank. However, because world_alcohol currently has a unicode datatype, all of the values in the column are strings.

To add these values together, or perform any other mathematical operations on them, we'll need to convert the column to floats.

Before we can convert the column to floats, we have to deal with empty string values (''). These values were inserted in the original data file when there was no data for the given country and year. If we try to convert the column to floats without removing these values first, we'll get a ValueError.

Thankfully, we can remove these items using the replacement technique we learned in the last screen.

7: Converting data types

We can convert the data type of an array using the astype() method. Here's an example:

xxxxxxxxxx
 
vector = numpy.array(["1", "2", "3"])
vector = vector.astype(float)

The above code will convert vector to all float values -- [1.0, 2.0, 3.0].

We'll need to convert the fifth column of world_alcohol to float values if we want to perform any computations. 8: Computing with NumPy

Now that alcohol_consumption consists of numeric values, we can perform computations on it. NumPy has a few built-in methods that operate on arrays. You can view all of them in the documentation, but here are a few important ones:

  • sum() -- computes the sum of all the elements in a vector, or the sum along a dimension in a matrix.
  • mean() -- computes the average of all the elements in a vector, or the average along a dimension in a matrix.
  • max() -- computes the maximum of all the elements in a vector, or the maximum along a dimension in a matrix.

Here's how we could use one of these methods on a vector:

xxxxxxxxxx
 
vector = numpy.array([5, 10, 15, 20])
vector.sum()

The code above would add together all of the elements in vector, and result in 50.

With a matrix, we have to specify an additional keyword argument axis. The axis dictates which dimension we perform the operation on. 1 means that we want to perform the operation on each row, and 0 means on each column. Here's an example where an operation is performed across each row:

xxxxxxxxxx
 
matrix = numpy.array([
                [5, 10, 15], 
                [20, 25, 30],
                [35, 40, 45]
             ])
    matrix.sum(axis=1)

The code above would compute the sum for each row, resulting in [30, 75, 120].

10: Calculating consumption for each country

Now that we know how to calculate the alcohol consumption for a single country and year, we can scale up the process to calculate consumption for all countries in a given year. Here's a rough process:

  • Create an empty dictionary called totals.
  • Select only the rows in world_alcohol that match a given year. Assign the result to year.
  • Loop through a list of countries. For each country:
    • Select only the rows from year that match the given country. Assign the result to country_consumption.
    • Extract the fifth column from country_consumption.
    • Replace any empty string values in the column with the string 0.
    • Convert the column to the float data type.
    • Find the sum of the column.
    • Add the sum to the totals dictionary with the country name as the key.
  • At the end, you'll have a dictionary containing the name of each country as keys, with the associated total alcohol consumption as the values.

12: NumPy strengths and weaknesses

You should now have a good foundation in using NumPy, and in some of the challenges around transforming and working with data. NumPy is much easier to work with than lists of lists:

  • It's easy to perform computations on data.
  • Data indexing and slicing is faster and easier.
  • Data types can be converted quickly.

Overall, NumPy makes working with data in Python much more efficient. NumPy is widely used in a variety of tasks, particularly machine learning, due to this efficiency.

However, you may have noticed some limitations with NumPy as you worked through the past 2 missions:

  • A whole array has to be the same datatype, which makes it cumbersome to work with many datasets.
  • Columns and rows have to be referred to by number, which gets confusing when you go back and forth from column name to column number.

In the next few missions, we'll learn about the Pandas library, one of the most popular data analysis libraries. Pandas builds on NumPy but does a better job of addressing the limitations of NumPy.
